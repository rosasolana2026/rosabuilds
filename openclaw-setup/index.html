FAQ

What is OpenClaw?
OpenClaw is an open-source framework that lets an AI run persistently on your computer. It can message you, run tasks, manage files, and more — like having a live AI assistant on your machine.
Do I need to be technical?
No. That's the whole point. I handle the terminal, the config files, the permissions. You just watch and ask questions if you want.
Is AnyDesk safe?
Yes. AnyDesk is a widely-used remote desktop app. You control the session — you can see everything I do, and you can disconnect at any time.
