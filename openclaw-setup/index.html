FAQ

What is OpenClaw?
OpenClaw is an open-source framework that runs an AI persistently on your machine. It connects to Claude (or other models), watches for messages, runs tasks autonomously, and stays alive 24/7. Like having an AI employee who lives on your computer.
Do I need to be technical?
No. That's the whole point of the Remote Install and Cloud VM tiers. For DIY, you need to be comfortable with a terminal. For the paid tiers, you just watch. I handle everything.
What does the live terminal feed show?
When I'm installing remotely, you see a real-time log of every command I run ‚Äî package installs, config writes, permission grants, and the final WhatsApp QR code. Nothing hidden. You own your machine.
Is AnyDesk safe?
Yes. AnyDesk is a widely-used remote desktop app. You see everything I do in real time, and you can disconnect at any moment. I request view + control permissions only; you accept them explicitly.
How cheap is the Cloud VM really?
Our cost is ~$4/month per VM (Hetzner CX11 ‚Äî 2GB RAM, 2 vCPU, 40GB SSD). We charge $49/month because we do the setup, monitoring, restarts, and support. You're not paying for compute ‚Äî you're paying for it to just work.
Mac only?
Remote Install is Mac-first. The Cloud VM tier runs Linux and works for anyone regardless of local OS. Windows support for Remote Install is possible ‚Äî reach out.
