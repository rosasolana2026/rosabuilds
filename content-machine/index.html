Questions

What's the cost to run it?

About $0.01‚Äì0.05 per day in Claude API costs. You need an Anthropic API key (pay-as-you-go). That's it.

Does it actually sound like me?

It depends on how well you fill out the brand voice file. The more examples and specifics you give it, the better. The setup package includes me tuning this with you until it does.

What's OpenClaw?

An open-source AI agent that runs on your machine. Content Machine is built as an OpenClaw skill but works as a standalone script too ‚Äî no OpenClaw required.

Do I have to post everything it generates?

No. Content saves to a folder. You review, edit, approve. Nothing gets posted without your say unless you configure auto-scheduling.

Can it post for me automatically?

Yes ‚Äî connect upload-post and it'll schedule posts across Twitter, LinkedIn, and more. The setup package includes this configuration.
